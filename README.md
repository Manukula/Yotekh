Run the local Flask server and open the page.

1. (Optional) Create a virtualenv and activate it.

2. Install Flask:

```bash
pip install flask
```

3. Run the server:

```bash
python app.py
```

4. Open http://127.0.0.1:5000/ in your browser and click the buttons. The `Members` button runs `chat.py` and shows its output in the content box.
